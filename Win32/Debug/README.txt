Для установки необходимо:

1. Скопировать "STD_Player.exe", "PlayListManager" и директорию "Images" в папку программы (Например, C:\Player).
2. Создать папку с настройками
	2.1 В Windows XP C:\Documents and settings\[Пользователь]\Application Data\СтД\Player
	2.2 В Windows Vista, 7, 8 C:\Users\[Пользователь]\AppData\Roaming\СтД\Player
3. Скопировать "config.ini" в папку с настройками.
4. Отредактировать "config.ini":
	4.1 В разделе "start options"
		4.1.1 file - написать адрес реально существующего spl файла (Например, "Ревизор.spl")
		4.1.2 ListDir - Написать адрес директории с spl файлами (В КОНЦЕ ПОСТАВИТЬ "\")
	4.2 В разделе "debug":
		4.2.1 SetResulution - Изменить разрешение. Если у Вас не 1024х768, то рекомендуем поставить 1, в противном случае можно поставить 0.
		4.2.2 width - Ширина экрана, устанавливаемая SetResulution. 
		4.2.3 height - Высота экрана, устанавливаемая SetResulution.
		4.2.4 CloseButton - Рисовать ли в верхнем левом углу кнопку закрытия. По умолчанию она не выводится (Компьютер выключается кнопкой Power или кнопкой «Выключить компьютер» в приложении, а само приложение не должно выключаться по идеологии).
5. Для полноты картины можно поместить программу в ветку автозагрузки реестра, но это необязательно. В рабочем компьютере программа запускается именно так.
6. Настроить файлы spl
	6.1 Скопировать папки "ListDir" и "Music" в любое место.
	6.2	Открыть "PlayListManager.exe"
	6.3 Указать желаемый spl-файл.
	6.4 В настройках "Директория" указать путь к папке, где хранится указанная музыка.
	
7. Запустить "STD_Player.exe"
	7.1 За функции Play, Pause, Stop отвечают клавиши клавиатуры "s", "d", "f" для первой деки и "k", "l", ";" для второй соответственно.
	7.2 За функции вверх-вниз по списку отвечают клавиши "e", "x" для первой деки и "o", "," для второй соответственно.
	7.3 Все остальные клавиши на оригинальной клавиатуре убраны.
